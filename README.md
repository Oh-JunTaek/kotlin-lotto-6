4. [로또 게임 구현]
5. 1부터 45까지의 숫자를 중복되지 않게 선택한다.
6. 로또를 발행할 때 마다 중복되지 않는 숫자 6개를 뽑는다.
7. 로또 1장의 가격은 1천원
8. [당첨 기준 설정]
9. 1등 6개 번호 일치 - 2,000.000.000원
10. 2등 5개 번호 + 보너스 번호 일치 30,000,000원 ->보너스 번호 설정 방법
